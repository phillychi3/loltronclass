# TronClass 工具集合

## tronclass util

### 安裝

請至：https://greasyfork.org/zh-TW/scripts/481617-tronclass-util

### 簡介

這是一個 TronClass 的小工具集合，目前具有以下功能：

-   快速觀看影片
-   一鍵完成每周影音教材觀看
-   一鍵讀取投影片
-   優化課程活動頁面
-   優化課程教材頁面
-   優化作業教材頁面
-   隱藏頁尾
-   防止影片焦點偵測（背景播放不中斷）
